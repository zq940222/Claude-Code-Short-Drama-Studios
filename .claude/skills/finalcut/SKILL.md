# /finalcut 剪映精剪

## 前置检查

- 本集全部镜头 status=success（粗剪 /edit 做过更好，但非必需——精剪直接用原始镜头）
- BGM 未生成时提示先跑 `/music`（也可选择无 BGM 继续，字幕和转场照做）
- 剪映已安装（精剪师会自查草稿目录，未装会停下给版本建议）

## 流程

1. 调度 **finalcut** agent：
   - 生成台词字幕 SRT（按 shotlist 时长累计 + 分镜表台词映射）
   - 写 pyJianYingDraft 脚本生成剪映草稿：视频轨（镜头序列+场间转场）、BGM 轨（按对位说明压低音量）、字幕轨、全局滤镜
   - 产出 `05-final/finalcut-ep{NN}.md` 精剪说明
2. 向用户报告草稿名称/位置，提示打开剪映检查三个常见微调点：转场、字幕断句、BGM 音量
3. 用户确认导出成片后，更新 project.json：status.final → done
4. 提示下一步：`/publish` 发布到短视频平台

## 注意

- 剪映版本约束：草稿生成对 5.9 支持最完整；自动导出仅 ≤6.8 支持；新版剪映草稿加密，失败时退回 /edit 粗剪 + 手动精剪
- 素材用绝对路径写入草稿，提醒用户别移动项目目录，否则剪映里素材丢失
