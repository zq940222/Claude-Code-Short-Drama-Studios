# /publish 平台发布（含门禁④）

## 前置检查

- 成片存在（剪映导出的成片或 05-final 粗剪均可，问用户用哪个文件发布）
- 浏览器已登录目标平台创作者中心（未登录时运营 agent 会停下等用户扫码）

## 流程

1. 调度 **operator** agent 产出发布物料到 `06-publish/ep{NN}/`：
   - `copy.md`：3 个候选标题、正文、话题标签、发布时段建议、置顶评论话术
   - `cover.png`：封面图（成片抽帧 + 大字标题；剧照不够炸时走生成路径）
2. 用户选定标题和封面后，问发布方式：**立即发布 / 定时发布（要时间）/ 只出物料不发布**
3. 发布执行（半自动）：operator 用浏览器自动化上传视频、填文案、传封面
4. **门禁④**：全部填好后截图发布预览页，用 AskUserQuestion 让用户确认后才点发布
5. 发布后截图存档，写 `publish-log.md`（时间、链接、终稿文案）；多平台逐个执行，每个平台单独过门禁④

## 硬性规则

- 未经门禁④确认绝不点发布；账号登录永远由用户本人完成
- 连载剧提醒保持标题格式和封面版式的系列一致性（参考上一集的 publish-log）
